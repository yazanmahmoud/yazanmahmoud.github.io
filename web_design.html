Web Design Project

As part of my college degree requirements, I completed a work placement with Creative Logic Consulting, where I developed a functional mock version of the Fultz House Museum website. My work involved cleaning and updating HTML code, fixing broken links, and making minor enhancements to the existing CSS for better structure and usability. I then built the application using React.js and JavaScript, and implemented AI-enhanced features, including real-time time, date, and weather functionality. The project was pushed to a new GitHub repository and successfully deployed using GitHub Pages.

The live project link is: https://yazanmahmoud.github.io/myfultzapp
